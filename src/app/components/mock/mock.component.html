Mock Exam

I created a web based Mock Exam site for my client. It is designed to recreate the digital exam experience and will be used for coaching CFA Examination Candidates.
I also created a Question Bank program for Exam Administrator to manage the exam syllabus and questions, as well as to easily create exam sessions for the Mock Exam site to use.
This is my second freelance project and I am currently maintaining it.   (2020)

Technologies used: Angular, Node.js, Express, MongoDB, jQuery, and EJS.
The source code for this project is not public.
